# RNA-seq Analysis Pipelines

R pipelines for transcriptomics data analysis from a stem cell differentiation project towards extra-embryonic lineages under BMP treatment. The repository contains two independent scripts: one for **bulk RNA-seq** and one for **single-cell RNA-seq (scRNA-seq)**.

---

## Input data

### Bulk RNA-seq (`Bulk_ARN_seq.R`)

| File | Description |
|---|---|
| `data/Elena*.tsv` | Expression tables (TPM) generated by STAR |
| `data/transcripts_quant_Elena*/quant.sf` | Salmon quantification files |
| `data/gene_count_matrix.csv` | Integer count matrix generated with StringTie's `prepDE.py` |

The expected columns in the STAR `.tsv` files are `Gene.ID` and `TPM`. The count CSV must have `gene_id` as the first column.

**Samples and conditions:**

| Internal name | Condition |
|---|---|
| WT1, WT2 | mTeSR (control) |
| BMP_0_48_1, BMP_0_48_2 | BMP4 10 ng/mL, 0–48 h |

### Single-cell RNA-seq (`single_cell.R`)

| File | Description |
|---|---|
| `filtered_feature_bc_matrix.tar.gz` | Cell Ranger output (10x Genomics) |
| `full_annotation.csv` | Cell type projection from the Human Embryo Atlas |

The annotation file must contain the columns `query_cell` and `sub_pred_EML`.

**Samples and conditions (6 multiplexed libraries):**

| ID (`orig.ident`) | Condition | Time point |
|---|---|---|
| 1 | BMP2.5 | 0–30 h |
| 2 | BMP10 0–16 h + mTeSR 0–30 h | 30 h |
| 3 | BMP10 | 0–30 h |
| 4 | BMP2.5 | 0–48 h |
| 5 | BMP10 0–16 h + mTeSR 0–48 h | 48 h |
| 6 | BMP10 + Noggin 30–48 h | 48 h |

## Usage

### Bulk RNA-seq

Open `Bulk_ARN_seq.R` and update the file paths near the top of the script:

```r
prefix <- "path/to/your/data/folder"   # Line ~58
```

Then run the full script in RStudio or from the terminal:

```bash
Rscript Bulk_ARN_seq.R
```

### Single-cell RNA-seq
run:

```bash
Rscript single_cell.R
```

> **Memory note:** The scRNA-seq script is configured with `options(future.globals.maxSize = 8000 * 1024^2)` (~8 GB). Running on a machine with at least 16 GB of RAM is recommended.

## Analysis description

### Bulk RNA-seq

1. **Mapper quality control** — comparison of STAR and Salmon quantifications using Venn/UpSet diagrams, PCA, and Spearman correlation scatter plots.
2. **Differential expression with DESeq2** — BMP4 vs. mTeSR (control) contrast using the StringTie/prepDE count matrix.
3. **Visualisation** — sample distance heatmap, most variable genes heatmap (top 40 by default), PCA, and a customisable volcano plot (adjustable `padj` threshold and number of labelled genes).

### Single-cell RNA-seq

1. **Data loading and Seurat object creation** — 10x data loading, initial filter (`min.cells = 3`, `min.features = 200`).
2. **Quality control** — mitochondrial and ribosomal percentage calculation, cell filtering (`nFeature_RNA > 4000`, `nCount_RNA < 100000`, `percent.mito < 10`).
3. **Reference annotation integration** — Human Embryo Atlas cell type projection onto filtered cells.
4. **Normalisation and variable genes** — LogNormalize + selection of 3000 variable features (VST), with manual addition of key marker genes (SOX2, NANOG, GATA3, CDX2, TBXT, etc.).
5. **Cell cycle correction** — S and G2/M phase scoring and regression of their effect in ScaleData.
6. **Dimensionality reduction and clustering** — PCA (19 PCs selected by elbow plot), 2D and 3D UMAP, Leiden clustering (resolution 1.0, algorithm 4).
7. **Visualisation** — UMAP coloured by cluster and experimental condition, violin plots and feature plots of key markers, dot plot, interactive 3D UMAP with Plotly, cell proportion bar charts per condition.
8. **Manual cluster annotation** — 25 clusters assigned to lineages: amnion, ExEM (extra-embryonic mesoderm), pluri (pluripotent), pluri_PS, PS_mesoderm, and ambiguous.

## Biological context

The data come from a human pluripotent stem cell (hPSC) differentiation experiment towards extra-embryonic lineages under different BMP treatment conditions (BMP2.5 and BMP10) at two time points (30 h and 48 h), with and without Noggin. The goal is to characterise the lineage specification dynamics of amnion, extra-embryonic mesoderm (ExEM), and pluripotent cells during early gastrulation.
